### 写一个方法将下面 arr 中的数据去重，去重后的结果应为result所示结果


```js
const arr = [
    'null',
     null,
     null,
     undefined,
     {},
     {name: 'xiaoyu'},
     {name: 'xiaoyu'},
     {title: 'xiaoyu'},
     {name: undefined},
     {}
];
const result = ['null', null, undefined, {}, {name: 'xiaoyu'}, {title: 'xiaoyu'}, {name: undefined}];
```
